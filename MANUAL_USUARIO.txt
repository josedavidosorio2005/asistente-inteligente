MANUAL DE USUARIO - ASISTENTE INTELIGENTE

REQUISITOS:
- Python 3.8 o superior
- Instalar dependencias: pip install -r requirements.txt
- Archivo de credenciales Google Drive: credentials.json (descargar desde Google Cloud Console)

USO EN OTRA PC:
1. Clona o copia la carpeta del asistente a la nueva PC.
2. Instala Python si no está instalado.
3. Abre terminal en la carpeta del asistente.
4. Ejecuta: pip install -r requirements.txt
5. Coloca tu archivo credentials.json en la carpeta raíz.
6. Ejecuta el asistente: python asistente_mic.py
7. La primera vez que sincronices con Drive, se abrirá una ventana para autorizar tu cuenta Google.

FUNCIONES PRINCIPALES:
- Interfaz moderna, comandos por voz en español.
- Crear, editar, buscar y eliminar notas por voz.
- Sincronizar notas con Google Drive:
    - "Sube notas a Drive" → Sube solo las notas nuevas.
    - "Descarga notas de Drive" → Descarga solo las notas nuevas.
    - "Sincroniza notas con Drive" → Sube y descarga.
- Abrir apps, buscar en Google, reproducir música, controlar el sistema, etc.

NOTAS:
- Las notas se guardan en la carpeta 'notas'.
- El archivo token.pickle se genera automáticamente tras autorizar Google Drive.
- Si cambias de PC, copia también 'notas' y 'token.pickle' para mantener tus datos.

SOPORTE:
- Si tienes errores de credenciales, revisa que 'credentials.json' sea válido y esté en la carpeta correcta.
- Si tienes problemas de dependencias, ejecuta de nuevo: pip install -r requirements.txt
